
Built with Next.js 16 (Latest) | Tailwind CSS | Microsoft Clarity
A high-performance, SEO-optimized, and cinematic personal portfolio designed to showcase full-stack engineering capabilities. This project focuses on Core Web Vitals, Smooth UX, and Data-Driven Insights.

🛠 Technical Architecture
Framework: Next.js 16.0.10 — Leveraging the latest App Router features and Server Components.

Styling: Tailwind CSS — For a highly responsive and utility-first UI design.

Smooth Scroll: Lenis 1.3.15 — Implementing high-end cinematic scrolling dynamics.

Analytics: Microsoft Clarity — Integrated for heatmaps, session recordings, and user behavior analysis.

Icons: Lucide React — For clean, consistent, and lightweight vector iconography.

Security: HSTS (HTTP Strict Transport Security) — Enforcing secure server connections.

Deployment: Vercel — Optimized for edge delivery and CI/CD workflows.

✨ Performance & Optimization
Priority Hints: Implemented fetchpriority on critical LCP (Largest Contentful Paint) elements to ensure lightning-fast visual loading.

Advanced Rendering: Utilized a hybrid of Static Site Generation (SSG) and Server-Side Rendering (SSR) for optimal speed and SEO.

Smooth Motion: Integrated Lenis to bridge the gap between native scrolling and high-end creative web experiences.

Responsive Architecture: A mobile-first approach ensuring 100% compatibility across all device viewports.
