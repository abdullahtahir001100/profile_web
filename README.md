## Tech Architecture  

- Next.js (App Router, Server Components)  
- Tailwind CSS (Utility-first UI)  
- Lenis (Smooth scrolling engine)  
- Microsoft Clarity (Analytics & heatmaps)  
- Lucide React (Icons system)  
- HSTS (Security headers)  
- Vercel (Deployment & CI/CD)
